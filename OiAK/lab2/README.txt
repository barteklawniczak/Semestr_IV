Program obsługuje mnożenie liczb heksadecymalnych poprzez wielokrotne
dodawanie wczytanej mnożnej.
Gdy wczytany mnożnik jest bardzo duży (6 liczb hex), wyświetlany jest komunikat core dumped.
Program sprawdza poprawność ciągu wpisanego przez użytkownika, zarówno pod względem długości jak i znaków w ciągu.
